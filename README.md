# quick http-server
Minimum http server setup in Node.js/Express for your Angular app

## Steps:
1. <code>cd http-server && mkdir public && cd public</code>
2. Copy your AngularJS entry file <code>index.html</code> & front end resources to the 'public' folder
1. <code>npm install && npm start</code>
4. Your app will start at <code>http://localhost:8080</code>. Your  static assets will be served automatically by Express.
